Addition

Properties of Addition

Identity
A + 0 = A
Commutative
A + B = B + A
Associative
(A + B) + C = A + (B + C)

Examples

  • 1 + 1 = 2
  • 2 + 2 = 4
  • 3 + 4 = 7
  • 12 + 23 = 35
