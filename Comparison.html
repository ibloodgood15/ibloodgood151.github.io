HTML and CSS

Since we have looked at both HTML and CSS in Web Design this year, let's take a moment to compare them. Here are some similarities and differences between HTML and CSS.

Similarities

Opening and saving

  • Both HTML and CSS are saved as plain text and can be typed using a simple program like Notepad.
  • HTML and CSS are usually saved in the same directory (or folder) so that they can work together.
  • Both HTML and CSS can be accessed on the Internet using HTTP.

What is in them

  • HTML and CSS understand the same elements (h1, br, etc.).
  • Both HTML and CSS provide ways for you to put in comments (notes for yourself) in the file that do not affect how the final page looks.
  • Both use the hash symbol (#) to represent an ID to look up.

Formatting

Some older, deprecated HTML elements overlap in purpose with CSS.

  • The <b> element makes things bold, like font-weight: bold.
  • The <small> element makes things smaller, like font-size: smaller.
  • The <center> element (officially gone now, but it probably still works) centers text horizontally, like text-align: center.

Differences

Punctuation

HTMLCSS
Uses angle brackets (< and >) Uses curly braces ({ and })
Starts a section with an opening tag (like <a>) and ends it with a closing tag (like </a>) Starts a section with a selector and an opening curly brace (like a{) and ends it with just a closing brace (})
Attributes are set with an equal sign and a value in double-quotes Properties are set with a colon followed by the value, not necessarily in quotes

Purpose

HTMLCSS
HTML is currently designed to contain content. CSS is designed to format existing content.
HTML can be used by itself. CSS only makes sense when used in combination with HTML.
HTML has changed purpose over the years. At one point it also included some formatting elements before CSS was introduced. CSS has always been about formatting - nothing else.

History

HTMLCSS
HTML's syntax and punctuation were inspired by the more general-purpose SGML language. CSS's syntax and punctuation appear to be inspired by programming languages like C, C++, Java, and JavaScript.
HTML was originally developed by Tim Berners-Lee around 1990. CSS was developed by Håkon Wium Lie and Bert Bos around 1994 to 1996.
